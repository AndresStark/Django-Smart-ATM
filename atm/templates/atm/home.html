Django ATM - Smart Automatic Teller Machine Web Interface

A client-based ATM simulation system built with Django, mimicking the core functionalities of an Automatic Transfer Machine (ATM). Users can authenticate, view their balance, withdraw or deposit money and view transaction history. Admins can manage user accounts and ATM cash balance

Applications

Automatic Teller Machine main functions. These applications can connect to an external API, serving as interface-less application as well
